# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/newshub.git
cd newshub
```

## Install Dependencies

```bash
npm install
```

OR install manually:

```bash
npm install react-icons
npm install react-leaflet
npm install leaflet
npm install leaflet.heat
```

## Run Development Server

```bash
npm start
```
